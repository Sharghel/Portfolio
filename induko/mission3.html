Mission 3

Intégration d'une BDD en php

Avec la base d’un template nous avions pour but de créer du code php pour incrémenter un tableau en html. Cet avec ce projet que nous avions pu voir les fondamentaux de php ainsi que les prémices des classes.

Description des tâches :
Pour la création de ce site web j’ai dû, développer mes connaissances en PHP :
  • Les bases du PHP
  • Traitement des requêtes http en get et post
  • Visions sur les classes
En SQL :
  • Création de requêtes préparée
En Sécurité :
  • Utilisation de la PDO
  • Faire des tests contre les injections
Divers :
  • Comment développer en suivant une maquette
  • Organisation d'un fichier (découpage, ordre)
Mes différents outils utilisés :
Logiciel principal :
  • VSCode pour le code en général
  • Figma pour l’accès à la maquette

J'ai fait ce projet en grande parti seul mais j’ai pu être aidé par mon intervenant ou d’autre élève quand j’avais des questions.
Le projet à été commencé le 2022 et fini le 2023. En tout nous avions travailler dessus l'équivalent d'une semaine
