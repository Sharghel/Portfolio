Mission 5

Création d'un portfolio

Pour la création d’un portfolio nous étions plus ou moins libre sur comment nous pouvions le faire, que ce soit sur Wordpress, Wix, Odoo, via Framework, suivant une template ou même en natif si l’on souhaite.
Me concernant j’ai pris le choix de le faire en natif en me basant depuis un template. Ce choix ne m’a pas freiné dans la création de ce site pour autant car j’ai quand même souhaité à apporter des améliorations pour rendre le site + agréable.

Description des tâches :
Pour la création de ce site web j’ai dû apprendre à refaire du HTML :
  • Revoir un code déjà existant, le comprendre et l'améliorer
  • être capable de pouvoir simplifier le code déjà existant
En Ajout :
  • Rajouter des parties importantes (scroller dynamiques)
Divers :
  • Acheter un nom de domaine
  • Paramètrer un server pour herger le site
  • Sécuriser le site avec un protocole ssh
Mes différents outils utilisés :
Logiciel principal :
  • VSCode pour le code en général
  • Figma pour l’accès à la maquette
Site externe :
  • Site Microlead avec les sections à propos de HTML et CSS

J'ai fait ce projet seul.
Le projet à été commencé en 2023 et fini en 2024 avec environ 50Heures de travail cummulé.
